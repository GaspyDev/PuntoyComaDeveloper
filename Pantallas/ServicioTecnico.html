En esta sección encontraras mas informacion referente a los servicios que proporcionamos enlistando los diferentes pasos que se hace en cada uno de ellos

Mantenimiento Preventivo
  • Se quita la carcasa del dispositivo con cuidado cuidando los tornillos y componentes con los que esta sujeto
  • Se desconecta la fuente de energia para evitar cortos mientras se manejan los componentes
  • Se desarman las partes correspondientes tales como: Disco duro, Tarjeta de Red, Disipador de calor, Ventilador, las bocinas, etc...
  • Quitados todos los componentes se procede a realizar limpieza con alcohol isopropilico para evitar corto circuito con los componentes
  • Se quita el exceso de polvo en el interior de la laptop
  • Se recomienda hacer el cambio de la pasta termica al menos una ves por año para mantener la temperatura de manera eficiente
  • Una ves terminada se ensambla todo de nuevo y se corrobora que funciona de manera correcta
Mantenimiento Correctivo
  • Se hace un disgnostico previo listando los posibles problemas que pudieran causar la falla
  • Una ves terminada la lista se procede a realizar el descarte de cada uno segun corresponda
  • Encontrada la falla se procede a realizar las correccciones pertinentes
  • Se determina si se requiere reemplazo de alguna pieza si es asi se le da aviso
  • Adquirida la pieza ya sea por nosotros o el cliente se reemplaza
  • Se corrobora que todo funcione correctamente y se hace la entrega al cliente
Formateo de Sistema Operativo
  • Se realizan los preparativos tales como respaldo de informacion importante para el cliente, enlistar programas que se tienen y se corrbora que el equipo funciona correctamente antes de iniciar
  • Se determina si se instalara nuevamente el mismo Sistema Operativo o sera reemplazado por otro
  • Se realiza el procedimiento correspondiente para el formateo
  • Una ves terminado se instala nuevamente todos los programas y drivers necesarios para el funcionamiento
  • Se corrobora que todo funciona correctamente y se entrega al cliente
Reemplazo de Componentes
  • Se realizan los preparativos para el reemplazo del componente asegurandonos de que el mismo sea compatible con el dispositivo
  • Se abre el equipo de computo para acceder al componente a reemplazar
  • Se limpia tanto el puerto donde va a ir el nuevo componente asi como el componente mismo
  • Se realiza el cambio ya sea de pantalla, bocinas, memoria Ram, tarjeta de Red, Procesador(en caso de que se pueda), Tarjeta de video
  • Se corrobora que el nuevo componente funciona correctamente dentro del sistema
Asesoria en Programacion

De igual manera si se requiere alguna clase particular para la comprension de algun tema de programacion en algun lenguaje especifico se puede brindar la atencion necesaria

  • Se determina el lenguaje y el tema a tratar durante la clase
  • Se acuerda un lugar, dia y hora que una ves acordados no podran ser reemplazados
  • Se dara la asesoria dependiendo del numero de horas acordadas
  • Se daran explicacion, ejemplos y ejercicios practicos en relacion al tema y tambien se realizaran prpeguntas para garantizar la correcta comprension del tema
